# TO-DO

A Simple, Clean **To Do App** Built Using **HTML,CSS AND JAVASCRIPT**.
Here Users can Add their tasks and work with them making their day easier.
---
## Features
- Add new Tasks.
- Mark Them Completed.
- Delete Them
- Review Completed Tasks
- Check the Progress (Tasks yet to do)

## Built With
- HTML
- CSS
- JavaScript
